VRfabeto

Bem vindo!

O projeto VRfabeto foi realizado como objetivo apresentar o trabalho/avaliação referente a matéria de Engenharia de Software ministrada pelo Professor Giuliano - FATEC SJC

Este site foi projetado para ajudar na alfabetização, utilizando o método silábico de alfabetização.
Ao escolher um dos objetos abaixo e utilizando dos arquivos disponiveis na area de "Materiais" as palavras serão materializados na tela do celular aguardando o usuário a lê-las em vóz alta, quando o usuário dizer a sílaba corretamente esta sinalizará com a cor verde, ao finalizar a palavra você poderá volta a página inicial
